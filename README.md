# HCU-Cut CityGMl Converter

## Getting Started

### Prerequisites

Install the requiremnts

```bash
python3 -m pip install -r requirements.txt
```

### Downloading Data

To get started download and extract the citygml dataset:

```bash
./download_gml.sh
```

### Convert Data

run the conversion script to get geojson (in './data/out')

```bash
python3 convert.py
```
